# Git hooks

Чтобы перед каждым коммитом автоматически обновлялись контрольные суммы итогового PDF в README, подключите локальные hooks:



=== "Task"


    ```bash
    task hooks
    ```



=== "Ручной"


    ```bash
    git config core.hooksPath .githooks
    ```





Для работы hook нужен Python-пакет `python-dotenv`. Он указан в `requirements.txt`.

Если окружение еще не подготовлено, установите зависимости:



=== "Task"


    ```bash
    task deps
    ```



=== "Ручной"


    ```bash
    pip install -r requirements.txt
    ```





Hook считает хэши текущего PDF алгоритмами из стандартного `hashlib`.[^pdf-hash] Если PDF отсутствует, README не меняется и коммит продолжается со старым значением.

[^pdf-hash]: Контрольные суммы фиксируют конкретный бинарный PDF, а не смысловое состояние исходников. Даже небольшие изменения метаданных или порядка сборки могут поменять хэш без заметной визуальной разницы.
