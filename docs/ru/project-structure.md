# Структура проекта

| Путь | Назначение |
| --- | --- |
| `*.tex` | Основной LaTeX-документ |
| `preamble/` | Настройки преамбулы |
| `bibliography.bib` | Dummy-библиография |
| `scripts/` | Выбранные вспомогательные скрипты |
| `tasks/` | Task-команды |
| `docs/` | Документация, если включена |

Листинги кода зависят от `code_listing_mode`: при `inline` код вставляется
вручную через окружения `lstlisting`, при `external` LaTeX может ссылаться на
отдельный каталог с исходниками.
