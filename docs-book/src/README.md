# Исходники диплома

Репозиторий содержит исходники дипломной работы: `LaTeX`-документы, `Mermaid`-диаграммы, Python-диаграммы, DOCX-шаблоны титульных страниц и Docker-профили для воспроизводимой сборки.

Основной `README.md` можно держать коротким: быстрый старт, ссылки и текущее состояние проекта. Подробные инструкции вынесены в эту книгу.

## Основной сценарий

Установите Task:

```powershell
winget install Task.Task
```

Для macOS или Linux с Homebrew:

```bash
brew install go-task/tap/go-task
```

Если эти варианты не подходят, используйте официальную инструкцию установки: <https://taskfile.dev/docs/installation>.

Проверить установку:

```bash
task --version
task --list
```

```bash
task build:images
task build
```

Максимальная воспроизводимость с оригиналом будет, если собирать `Mermaid`-диаграммы из-под Windows. При сборке Mermaid через Docker шрифт для `KaTeX`-выражений может отличаться от оригинала.

Для запуска основного сценария нужен Docker. Если Docker не используется, см. раздел [Сборка без Docker](manual-build.md).
