# Документация mdBook

Подробную документацию можно хранить отдельно от README в `docs-book/`. Папка находится внутри репозитория, поэтому остается приватной вместе с ним.

## Локальный запуск

Запустить документацию:

```bash
docker compose --profile docs up
```

После запуска книга доступна в браузере:

```text
http://localhost:3000
```

Остановить сервис:

```bash
docker compose --profile docs down
```

Заранее скачать Docker-образ документации:

```bash
docker compose --profile docs pull
```

## Образ

Для документации используется компактный образ:

```text
peaceiris/mdbook@sha256:a82fd90af897238521c17e4b64479f5d56b578e4776198788256086b3a59617a
```

Фактический размер образа при проверке: `28.5 MB`.

Версия mdBook внутри образа:

```text
mdbook v0.5.0
```

Самосборный образ на базе Rust был больше: `99.5 MB`. Поэтому для локального сервера документации используется `peaceiris/mdbook`.

## Приватность

Исходники книги лежат в приватном репозитории и остаются приватными вместе с ним. Не публикуйте сгенерированную папку `docs-book/book/` в публичный хостинг, если документация должна оставаться закрытой.

GitHub Pages может требовать отдельной настройки приватного доступа. Для этого проекта базовый безопасный вариант - держать книгу в репозитории и запускать ее локально через Docker.
