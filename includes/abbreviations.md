*[Docker]: Платформа для запуска приложений в изолированных контейнерах с заранее описанным окружением.
*[Docker Compose]: Инструмент для запуска нескольких Docker-сервисов из одного файла `docker-compose.yaml`.
*[LaTeX]: Система подготовки документов, которая собирает исходный `.tex` файл в PDF.
*[LuaLaTeX]: Движок LaTeX на базе LuaTeX; в проекте используется как основной компилятор.
*[latexmk]: Утилита, которая сама запускает LaTeX и библиографию нужное количество раз.
*[biblatex]: LaTeX-пакет для работы с библиографией.
*[biber]: Backend для `biblatex`, который обрабатывает библиографические данные.
*[Mermaid]: Текстовый формат для описания диаграмм, которые можно рендерить в изображение или HTML.
*[KaTeX]: Библиотека для отображения математических выражений в браузере.
*[Zensical]: Генератор статической документации, который собирает Markdown-страницы в сайт.
*[Markdown]: Легкий текстовый формат разметки для документации.
*[DOCX]: Формат документов Microsoft Word, используемый здесь для титульника и задания.
*[PDF]: Итоговый формат диплома и промежуточный формат титульных страниц и диаграмм.
*[Git]: Система контроля версий, которая хранит историю изменений проекта.
*[GitHub]: Платформа для хранения Git-репозиториев и просмотра файлов проекта.
*[Task]: Утилита запуска проектных команд из `Taskfile.yml`.
*[diff-pdf]: Инструмент для визуального сравнения двух PDF-файлов.
