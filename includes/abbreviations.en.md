*[Docker]: A platform for running applications in isolated containers with predefined environments.
*[Docker Compose]: A tool for running multiple Docker services from one `docker-compose.yaml` file.
*[LaTeX]: A document preparation system that builds a source `.tex` file into a PDF.
*[LuaLaTeX]: A LaTeX engine based on LuaTeX; this project uses it as the main compiler.
*[latexmk]: A utility that runs LaTeX and bibliography tools as many times as needed.
*[biblatex]: A LaTeX package for bibliography management.
*[biber]: The backend for `biblatex` that processes bibliography data.
*[Mermaid]: A text format for describing diagrams that can be rendered to images or HTML.
*[KaTeX]: A library for rendering math expressions in the browser.
*[Git]: A version control system that stores the project's change history.
*[Task]: A command runner that reads project commands from `Taskfile.yml`.
*[diff-pdf]: A tool for visual comparison of two PDF files.
